# snhdb

A library of research papers, kept in one public repository so anyone can clone it and search the collection locally.

## Usage

```sh
git clone https://github.com/richbodo/snhdb.git
```

Then search with your tool of choice, e.g.:

```sh
grep -ri "your query" .          # text files
rga "your query" .               # ripgrep-all also searches inside PDFs
```

## Contents

Papers are added periodically. Organization and an index will evolve as the collection grows.
